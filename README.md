🍽️ Restaurante App - Sistema de Pedidos em Java

Este projeto é uma aplicação simples desenvolvida em Java, que simula um sistema de pedidos de restaurante via terminal. O sistema permite que o usuário visualize um cardápio completo de comidas e bebidas, escolha seus itens e receba um resumo detalhado do pedido com o valor total.

🚀 Funcionalidades
Exibição de menu com 25 opções de comidas
Exibição de menu com 25 opções de bebidas
Seleção de itens através de entrada do usuário
Validação de opções inválidas
Cálculo automático do valor total do pedido
Exibição de resumo completo (comida, bebida e total)
🛠️ Tecnologias Utilizadas
Java (JDK 8 ou superior)
Classe Scanner para entrada de dados
Estruturas de dados como arrays (String[] e double[])
Controle de fluxo com for e while
💡 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais da programação em Java, como:

Manipulação de arrays
Entrada e saída de dados
Estruturas de repetição
Validação de dados
Lógica de programação
